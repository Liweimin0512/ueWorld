# unWorld

Developed with Unreal Engine 4

这是一个用UE4 开发的3DRPG游戏
这个项目将包括一切RPG项目中常见的功能模块
以及你想看到的内容（欢迎给我留言）

## 开发环境

UE4.26
[UE4 vectory 插件](http://www.mediafire.com/file/0q10k9k3lumkbsk/VictoryPlugin26.zip/file)
基于GameplayAbilitySystem插件开发的属性系统
